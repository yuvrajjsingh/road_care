## About

Road Care is a web application that allows users to submit feedback about road conditions.  
It stores the feedback in a database and provides a simple interface to view reports.  
This project demonstrates full-stack web development skills using Python and Flask.

---

## Features

- User-friendly feedback submission form  
- View reports of road conditions  
- Simple and clean HTML interface  
- Uses Flask for backend and SQLite for database
